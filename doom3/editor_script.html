Making DOOM 3 Mods : The Script Editor

The script editor should not be launched with "editScripts". It should only be launched by using "editDecls", selecting a script, then clicking Edit.

The script editor is just like the decl editor except it does syntax highlighting, command completion, and has function declaration tool tips (like visual studio).

There's really not a whole lot to say about the script editor, it's pretty simple to use and understand.
