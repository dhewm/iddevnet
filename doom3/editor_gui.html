Making DOOM 3 Mods : The GUI Editor

The gui editor is a real life saver for making small, simple guis. Raven Software initially created the GUI editor for Quake 4 (which is why it says "Quake IV GUI Editor" in the title bar). The GUI Editor that is in Doom 3 hasn't been updated in a year or two, so when it randomly crashes, it's not their fault.
