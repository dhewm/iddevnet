This tab adjusts rotation values over the lifetime of a particle. (Not found on a Line segment)

Start – Min/Max

  • This is where the start rotation value of a particle is set. 

Example: If you wish for a particle to have a random rotation value when spawned in, but not rotate – set the Start Min value to 0 and the Start Max value to 360.

End – Min/Max

  • This is where the end rotation value of a particle is set.

End Relative to Start

  • End relative to start ensures that the end angle is relative to it's start angle. 

Example: A start value of 0/360. An end value of 20. If a sprite is chosen to start at 180, it will rotate to 200 rather than rotate to 20. If the end value is -20/20 and the sprite starts at 180, it will end in the range of 160-200.
  • With this setting unchecked, the rotation will always end at the exact end value rather than relative to it's start rotation value.

Envelope

  • Here an envelope can be applied to the rotation value. Please see Notes about Envelopes/Tables for more detailed information.
