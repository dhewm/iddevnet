What is this page?

This page contains a list of make-a-mod tutorials. They detail the steps, from start to finish, of modifying various parts of the game. Since modifications touch on so many different subsystems of the game, there are several make-a-mod tutorials listed.

The tutorials

  • MakeAMod-InstaGib - Modify def files to turn standard DM into a rail-only one-shot-one-kill game

  • MakeAMod-DMSP - Implement slews of monsters in a deathmatch map using only script and a slightly updated .map
