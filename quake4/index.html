News

Mon Jun 18 16:30:21 CDT 2007

The Quake 4 1.4.2 SDK is available from id Software ftp:

Windows 1.4.2 SDK installer Linux 1.4.2 SDK installer

Both installers have the same content. Pick one depending on your development OS. There is no Mac installer, but XCode project files are included in the above releases.

Wed Jan 07 16:26:32 CDT 2007

The Quake 4 1.4 SDK is available from the id Software ftp:

Windows 1.4 SDK Linux 1.4 SDK

Mon Jul 17 17:16:11 CDT 2006

We have opened a page dedicated to Benchmarking. We hope this will provide helpful information for anyone looking at benchmarking Quake 4 performance.

Tue Apr 18 00:41:40 CDT 2006

The SDK for the upcoming Quake 4 1.2 patch (1.1 + monitor fix) is now available from id Software's site. The SDK comes in Win32 and Linux flavors.

Tue Feb 21 14:52:57 CST 2006

GtkRadiant has been released under GPL. It supports level editing for Quake 4, and runs on Linux and Mac OS X as well. A great complement to the builtin Quake 4 editor!

Tue Feb 6 23:13:58 CST 2006

VIA CGA and Doom3World have announced the winners of their Quake 4 map competition and released a 12 map pack showcasing the best of their entries. Contest results and a brief overview of all levels included in the pack can be found here. Give them a shot and see what some others have been doing with the editor so far!

Wed Nov 16 11:14:46 CST 2005

The Quake 4 SDK is available from id Software's ftp. Expect to see it on mirrors and download sites within a few hours.

Documentation Updates

Thu Mar 16 10:51:08 CST 2006

Thanks to some clarification from id on their addonPak system (now implemented in Q4 1.1), we now have a definitive guide to setting up your mapDefs and addon.conf files for autodownload support. This also should serve as a good beginner's guide for anyone new to creating mapDefs for custom maps in Quake 4.

Wed Feb 22 16:11:42 CST 2006

Player model support has changed slightly with the release of the Quake 4 1.1 beta patch. We've added an overview of how to set up your custom models with the new .def file structure here. And yes, this means custom models can now be selected from the player selection menu!

Tue Feb 6 23:23:35 CST 2006

We've added a section detailing community resources available online in addition to this site. If you can't find what you're looking for here, or if you're looking for some community tutorials, tools, or forums, take a peek in CommunityResources (now linked below).

There's also an overview of NetworkDemos added from just a few days ago.
